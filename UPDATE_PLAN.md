# 更新计划 / Update Plan

- 先优先修复 bug，确保稳定性与可用性。/ First prioritize bug fixes to ensure stability and usability.
- 之后再逐步推进功能特性与体验优化。/ Then gradually roll out feature improvements and experience optimization.

2. 优化 UI，让界面更好看、更易用。/ Optimize the UI to make the app look better and be easier to use.
3. 新增单行的进度条图表与单行的纯数字展示图表。/ Add single-line progress-bar charts and single-line numeric-only display charts.
